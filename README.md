# Felise Atelier — Order & Fulfillment Management

## Quick Deploy (Vercel)

1. Go to **https://vercel.com** and sign up with your GitHub account
2. Click **"Add New..." → "Project"**
3. Import this repository
4. Click **"Deploy"**
5. Done! Your site is live.
